# ✈️ WanderLust — Туристичний веб-застосунок

Командний проект з Web-програмування (JavaScript)  
Тема: **Подорожі та туризм**

## 📁 Структура проекту

```
wanderlust/
├── index.html          # Головна HTML-сторінка (SPA)
├── css/
│   └── style.css       # Всі стилі (адаптивні)
├── js/
│   ├── app.js          # Логіка застосунку, AJAX, пошук, UI
│   └── game.js         # Аркадна гра «Лови Прапори»
└── data/
    └── destinations.json  # JSON-дані (завантажуються через fetch)
```

## 🌍 Функціонал

- **6 регіонів**, **25+ напрямків** з зображеннями та описами
- Привітання користувача за іменем (анімована реакція)
- Пошук із підсвічуванням результатів
- Модальні вікна з деталями кожного напрямку
- Адаптивний дизайн (5 брейкпоінтів)
- Аркадна гра «Лови Прапори» на Canvas
- Дані завантажуються через `fetch()` з JSON (AJAX)
- Navbar з hamburger-меню для мобільних

## 🎮 Гра «Лови Прапори»

Гравець керує літаком (← → або свайп) і ловить прапори країн, уникаючи важкого багажу. Є рівні, рахунок та система очок.
